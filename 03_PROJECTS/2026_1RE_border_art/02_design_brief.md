# Design brief — 2026_1RE_border_art

## 1. Principes appliqués

### 1.1 Backward Design

La séquence doit partir de la tâche finale et non d’une accumulation de supports. La tâche finale doit déterminer les besoins culturels, linguistiques, pragmatiques et méthodologiques.

Principe appliqué :
- définir d’abord ce que l’élève devra produire ;
- identifier les ressources nécessaires ;
- sélectionner ou hiérarchiser les supports en fonction de leur utilité pour cette production ;
- supprimer ou reléguer en option les supports qui ne préparent pas directement la tâche finale.

### 1.2 Perspective actionnelle

L’élève doit être placé dans une situation de communication vraisemblable. La tâche finale doit lui donner un rôle, un destinataire, un produit final et un objectif communicatif.

Application retenue :
- rôle : young curator / cultural project team ;
- destinataire : local cultural committee / school cultural committee ;
- produit : oral curatorial pitch ;
- but : convaincre que le Border Art peut aider le public à voir la frontière autrement, tout en reconnaissant la persistance possible des stéréotypes.

### 1.3 Accès au sens par action sur le document

Les activités devront éviter le simple questionnaire de compréhension. Les élèves devront agir sur les documents :
- observe;
- list;
- match;
- classify;
- highlight;
- justify;
- infer;
- recap.

La worksheet devra donc être un document de travail et non une trace écrite finale.

### 1.4 Gestion de la charge cognitive

Le corpus doit être resserré. La séquence ne doit pas devenir un catalogue d’œuvres ou d’initiatives autour de la frontière.

Principe appliqué :
- sélectionner un noyau réduit de supports ;
- utiliser les autres documents comme appoints, différenciation ou devoirs ;
- construire une progression claire : representation → stereotype → artistic reframing → curatorial pitch.

### 1.5 Adaptation sans dénaturation

Les documents authentiques doivent être rendus exploitables sans être simplifiés abusivement.

Le guidage devra passer par :
- chunking ;
- anticipation visuelle ;
- repérages ciblés ;
- scaffolding lexical ;
- tableaux de classement ;
- reformulations guidées ;
- class recap co-construit.

### 1.6 Progression spiralaire

Les mêmes fonctions langagières doivent revenir plusieurs fois dans des contextes différents :
- décrire une image ou une œuvre ;
- identifier une représentation ;
- nommer un stéréotype ;
- expliquer une intention artistique ;
- justifier un choix ;
- nuancer une opinion ;
- présenter un projet à un public.

## 2. Tensions / contradictions

### 2.1 Durée potentiellement longue

Une séquence de 7 séances peut être pertinente seulement si chaque séance apporte une brique directement utile à la tâche finale. Sinon, la séquence risque de s’étirer et de perdre sa tension.

Décision provisoire :
- conserver 6 séances + 1 séance d’évaluation si nécessaire ;
- éviter d’ajouter une séance uniquement documentaire ;
- intégrer la préparation de la tâche finale progressivement.

### 2.2 Problématique plus exigeante que le thème initial

Problématique stabilisée :

Can Border Art help us see the U.S.-Mexico border differently, or do stereotypes remain stronger?

Cette formulation est plus intéressante qu’une simple question sur l’art et le mur, mais elle impose de traiter explicitement la notion de stereotype.

Conséquence :
- la notion de stereotype doit apparaître dès l’ouverture ;
- chaque support doit aider à répondre à la problématique ;
- la tâche finale doit inclure une phrase de nuance sur les limites de l’art.

### 2.3 Risque de tâche finale trop événementielle

Un mini-festival ou une exposition peut facilement devenir une tâche de décoration ou d’organisation :
- title;
- poster;
- activities;
- programme;
- venue.

Ce serait insuffisant didactiquement.

Correction retenue :
- privilégier un curatorial pitch ;
- demander aux élèves d’expliquer le message culturel de leur mini-exhibition ;
- maintenir l’enjeu : changer le regard sur la frontière / questionner les stéréotypes.

### 2.4 Corpus riche mais dispersif

La richesse documentaire peut produire une séquence catalogue.

Correction retenue :
- un support d’ouverture pour les représentations ;
- un support de cadrage historique très court ;
- deux ou trois œuvres / projets artistiques maximum ;
- un temps explicite de préparation du pitch.

## 3. Inférences didactiques

### 3.1 Tâche finale

[inférence didactique]

La tâche finale la plus cohérente est un curatorial pitch plutôt qu’un exposé ou un festival général.

Formulation provisoire :

You are a team of young curators. Pitch a mini Border Art exhibition to a local cultural committee. Your aim is to show how your exhibition can help visitors see the U.S.-Mexico border differently — while acknowledging that some stereotypes may remain strong.

### 3.2 Structure argumentative attendue

[inférence didactique]

La production finale devra suivre une structure simple :

1. Present the problem: many people see the border as...
2. Present the exhibition: our project focuses on...
3. Explain the artworks: they show / challenge / reveal...
4. Justify the impact on visitors: it can help people understand...
5. Nuance: however, stereotypes may remain strong because...

### 3.3 Place de la notion de stereotype

[inférence didactique]

La notion de stereotype ne doit pas être seulement lexicale. Elle doit devenir un outil d’analyse culturelle.

Les élèves devront pouvoir dire :
- The stereotype is that...
- This artwork challenges the stereotype that...
- It presents the border as..., not only as...
- It helps visitors see migrants / border communities / the wall differently.

### 3.4 Évaluation

[inférence didactique]

L’évaluation devra porter sur une performance orale observable :
- clarté du projet ;
- lien explicite avec la problématique ;
- justification des choix artistiques ;
- capacité à nuancer ;
- organisation du pitch ;
- intelligibilité ;
- interaction courte.

## 4. Risques de conception

### 4.1 Séquence catalogue

Risque :
Accumuler des œuvres et exemples de Border Art sans progression intellectuelle.

Prévention :
Chaque support doit répondre à une fonction :
- ouvrir une représentation ;
- déplacer le regard ;
- complexifier ;
- outiller la tâche finale.

### 4.2 Worksheet de vérification

Risque :
Produire des worksheets composées de questions directes.

Prévention :
Préférer des tâches d’accès au sens :
- classify stereotypes / alternative representations;
- match artwork and message;
- highlight evidence;
- complete a curator’s note;
- justify one interpretation.

### 4.3 Trace écrite descriptive

Risque :
Produire une trace écrite du type “Border Art is beautiful and creative.”

Prévention :
Construire une trace écrite qui prépare la tâche finale :

Border Art can transform a wall into a message. It can challenge the stereotype that the border is only a place of danger or conflict. Some artworks present the border as a place of memory, community and connection. However, art may not erase stereotypes completely.

### 4.4 Tâche finale trop créative et pas assez langagière

Risque :
Évaluer surtout la créativité visuelle ou le design du support.

Prévention :
Le support visuel doit rester secondaire. La performance principale est le pitch oral argumenté.

### 4.5 Numérique décoratif

Risque :
Utiliser diaporama, Canva ou images comme décoration.

Prévention :
Le numérique doit servir :
- la médiation visuelle ;
- l’organisation du pitch ;
- la justification des choix ;
- la clarté du message culturel.

## 5. Design brief

### 5.1 Titre de travail

From Wall to Lens: Border Art and the Stereotypes of the U.S.-Mexico Border

### 5.2 Classe et niveau

Classe : Première  
Niveau visé : B1 → B1+  
Axe majeur : Identités et échanges  
Aire culturelle : United States / Mexico / borderlands  
AL dominante : EOC  
AL secondaires : CE, CO, EOI guidée

### 5.3 Problématique stabilisée

Can Border Art help us see the U.S.-Mexico border differently, or do stereotypes remain stronger?

### 5.4 Enjeu culturel

La frontière États-Unis-Mexique est souvent représentée comme une ligne de séparation, un mur, une zone de danger ou un espace de crise migratoire. Le Border Art propose d’autres représentations : mémoire, communauté, hybridité culturelle, lien, résistance et créativité.

La séquence doit amener les élèves à comprendre que l’art peut modifier la perception d’un territoire, mais que les stéréotypes peuvent rester puissants.

### 5.5 Tâche finale provisoire

You are a team of young curators. Pitch a mini Border Art exhibition to a local cultural committee. Your aim is to show how your exhibition can help visitors see the U.S.-Mexico border differently — while acknowledging that some stereotypes may remain strong.

### 5.6 Produit final

Curatorial pitch oral.

Format provisoire :
- binôme ou trio ;
- environ 3 minutes ;
- 1 support visuel simple ;
- 2 ou 3 œuvres / projets artistiques ;
- 1 message culturel clair ;
- 1 phrase de nuance ;
- courte interaction finale.

### 5.7 Objectifs culturels

Les élèves devront comprendre que :
- la frontière est un espace politique, humain et symbolique ;
- les représentations médiatiques peuvent produire ou renforcer des stéréotypes ;
- le Border Art peut transformer un mur en support d’expression ;
- les artistes peuvent créer du lien, de la mémoire ou de la résistance ;
- l’art peut déplacer le regard, mais pas nécessairement effacer les stéréotypes.

### 5.8 Objectifs linguistiques

Lexique prioritaire :
- border;
- wall;
- crossing;
- migrant;
- borderlands;
- community;
- identity;
- stereotype;
- artwork;
- mural;
- installation;
- exhibition;
- curator;
- audience;
- message;
- memory;
- connection;
- division.

Structures de service :
- to aim to;
- to intend to;
- to be designed to;
- It challenges the stereotype that...
- It helps visitors understand that...
- It presents the border as..., not only as...
- However...
- Whereas...
- Although...
- On the one hand...
- On the other hand...
- This artwork may change people’s perception because...

### 5.9 Objectifs pragmatiques

Les élèves devront apprendre à :
- décrire une œuvre ;
- interpréter une intention ;
- expliquer un effet sur un public ;
- justifier un choix curatorial ;
- organiser un pitch ;
- nuancer une position ;
- répondre brièvement à une question.

### 5.10 Objectifs phonologiques

À intégrer progressivement :
- accentuation des mots-clés : border, stereotype, community, connection, exhibition;
- groupes de souffle dans le pitch ;
- intonation persuasive ;
- accent de phrase sur les mots porteurs de message.

### 5.11 Supports à privilégier

Noyau recommandé :
1. document d’ouverture sur les représentations de la frontière ;
2. court cadrage historique ou visuel sur le U.S.-Mexico border wall ;
3. support pivot sur Border Art / Border Artivism ;
4. exemple d’exposition ou de projet artistique ;
5. exemple d’œuvre participative ou communautaire ;
6. banque de langue pour préparer le pitch ;
7. final project adapté en curatorial pitch.

Supports optionnels :
- documents trop riches ou trop périphériques à garder pour différenciation ;
- prolongements culturels à utiliser en devoir ou extension ;
- documents non directement utiles à la tâche finale à écarter du tronc commun.

### 5.12 Statut de la worksheet

La worksheet sera un outil de travail temporaire.

Elle doit permettre :
- repérage ;
- classement ;
- hypothèse ;
- justification ;
- préparation du recap ;
- préparation de la tâche finale.

Elle ne doit pas être :
- une trace écrite finale ;
- un questionnaire exhaustif ;
- une évaluation déguisée ;
- un document à remplir mécaniquement.

### 5.13 Statut du class recap

Le class recap doit être co-construit au tableau à partir des productions élèves, enrichi par le professeur, puis fixé dans le cahier.

Formulations utiles à construire :
- At first, the border may look like...
- This artwork challenges the stereotype that...
- It shows the border as...
- It helps visitors see...
- However, stereotypes may remain strong because...

### 5.14 Décision provisoire

La séquence est didactiquement prometteuse à condition de rester centrée sur la problématique des stéréotypes.

Décision de conception :
Garder le curatorial pitch comme tâche finale et organiser chaque séance comme une étape vers la réponse à la problématique :

Does Border Art really change the way we see the U.S.-Mexico border?
