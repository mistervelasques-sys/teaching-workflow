## 3. Principes appliqués

- Backward Design : cette séance installe l’argument central du futur pitch.
- Support pivot : `Border Artivism` sert à comprendre l’art comme action culturelle.
- Accès au sens par action : repérer, associer, classer, justifier, reformuler.
- Charge cognitive maîtrisée : chunking du support, pas d’exploitation exhaustive.
- Progression spiralaire : réactivation de `wall`, `stereotype`, `story`, `control`, `movement`.
- Injection linguistique juste-à-temps : `aims to`, `challenges`, `helps visitors understand`.
- Class recap co-construit : trace utile pour la tâche finale.

## 4. Tensions / contradictions

### Tension 1 — Support pivot dense vs niveau B1/B1+

`Border Artivism` peut être riche culturellement et lexicalement.

Décision :
- exploiter uniquement les éléments utiles à la tâche finale ;
- découper le support ;
- ne pas demander une compréhension exhaustive ;
- centrer sur action artistique → message → stereotype challenged.

### Tension 2 — Artivism comme notion abstraite

Le mot `artivism` peut rester opaque.

Décision :
- partir des deux composantes : art + activism ;
- faire associer actions artistiques et messages ;
- éviter une définition théorique longue.

### Tension 3 — Engagement politique vs tâche curatoriale

La séance ne doit pas devenir un débat politique général.

Décision :
- focaliser sur le rôle de l’artiste ;
- faire produire une phrase de médiation culturelle utile au pitch ;
- éviter les prises de position trop larges.

## 6. Objectif d’apprentissage

À la fin de la séance, les élèves seront capables de :

- comprendre que Border Art peut être une forme d’art engagé ;
- associer une action artistique à un message ;
- expliquer comment une œuvre peut contester un stéréotype ;
- produire une phrase courte réutilisable dans le curatorial pitch.

Formulation élève possible :

By the end of the lesson, I can explain how Border Art can transform the border into a message and challenge a stereotype.

## 7. Objectifs culturels

Les élèves comprennent que :

- le Border Art n’est pas seulement décoratif ;
- certains artistes utilisent la frontière comme support de message ;
- l’art peut dénoncer, relier, humaniser, questionner ou résister ;
- l’art peut transformer un mur ou une frontière en espace d’expression ;
- le Border Art peut aider les visiteurs à voir la frontière autrement.

## 14. Consignes prêtes à l’emploi

### Phase 1 — Reactivation

Complete orally:

The border is not only ___________________________.

It also tells a story of ___________________________.

### Phase 2 — Word building

Look at the word:

art + activism = artivism

What can you infer?

Artivism is probably art used to ___________________________.

### Phase 3 — First reading / spotting

Read the selected extract.

Highlight or list actions by artists.

Write key words only.

Examples:
- paint;
- create;
- transform;
- connect;
- denounce;
- question.

### Phase 4 — Action to message

Match one action with one possible message.

| Artistic action | Possible message |
|---|---|
| paint a mural | show that the wall can become a message |
| create a public artwork | make people think |
| involve a community | create connection |
| use the border as a canvas | transform division into expression |

### Phase 5 — Stereotype challenged

Complete orally:

This artwork challenges the stereotype that the border is only ___________________________.

It presents the border as ___________________________.

### Phase 6 — Micro-production

Complete one sentence:

This artwork aims to ___________________________.

It challenges the stereotype that ___________________________.

It helps visitors understand that ___________________________.

## 15. Tableau prévu

### Board step 1 — Reactivation

Lesson 1:
- first impression;
- stereotype;
- wall / division.

Lesson 2:
- the border is not only a wall;
- it tells a story of control, movement and human lives.

### Board step 2 — Artivism

art + activism = artivism

Artivism = art used to send a public message or support a cause.

### Board step 3 — Action / message / stereotype

| Artistic action | Message | Stereotype challenged |
|---|---|---|
| paint the wall | the wall can become a message | the border is only a place of division |
| involve people | communities have a voice | migrants are only victims |
| create a public artwork | art can make people think | the border is only dangerous |
| transform the wall | division can become expression | a wall is only a barrier |

### Board step 4 — Reusable sentence

Border Art can transform a wall into a message.  
It can challenge the stereotype that the border is only a place of danger or division.  
It helps visitors see the border as a place of expression, community and connection.

## 18. Différenciation

### Élèves fragiles

- Donner une banque de verbes :
  - paint;
  - create;
  - transform;
  - connect;
  - show;
  - question.
- Donner une banque de messages :
  - send a message;
  - create connection;
  - show community;
  - challenge stereotypes.
- Accepter une phrase simple :
  - This artwork sends a message.

### Élèves plus avancés

- Demander une justification :
  - I think this artwork challenges this stereotype because...
- Ajouter une nuance :
  - It may change visitors’ perception because...
- Leur demander de produire une phrase plus curatoriale :
  - By turning the wall into a public artwork, the artist invites visitors to reconsider the border.

## 19. Points de vigilance

- Ne pas faire une étude exhaustive du texte.
- Ne pas transformer artivism en cours théorique.
- Ne pas entrer dans un débat politique général.
- Ne pas multiplier les œuvres.
- Ne pas produire de worksheet maintenant.
- Ne pas perdre la chaîne : action → message → stereotype challenged.
- Garder le lien avec la tâche finale : future exhibition / visitors / perception.
