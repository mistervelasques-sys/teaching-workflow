# Audit didactique — 2026_1RE_border_art

## Verdict

Architecture globalement solide. La séquence est exploitable, mais **pas encore prête pour les lesson plans** tant que les supports exacts de chaque séance ne sont pas verrouillés.

Le point fort : l’alignement général est clair — représentations initiales → stéréotypes → Border Art → choix curatorial → pitch final.

Le point faible : plusieurs séances reposent encore sur des supports génériques (“document d’ouverture”, “support pivot”, “exemple d’exposition”) et non sur des documents précisément identifiés. Sans supports exacts, le risque est de produire des lesson plans artificiels.

## Sources consultées

- `03_sequence.md` — architecture globale produite.
- `02_design_brief.md` — cadrage validé.
- `Cadres méthodologiques.pdf` — Backward Design, perspective actionnelle, CECRL, distinction entraînement / vérification.
- `Activités d’entrée dans la thématique et d’accès au sens.pdf` — accès au sens par action sur le document : observer, repérer, classer, associer, justifier.
- `Adaptation du support.pdf` — chunking, anticipation, scaffolding visuel, adaptation sans dénaturation.
- `Approche spiralaire.pdf` — réactivation progressive, trace écrite comme référence, transfert dans des contextes nouveaux.

## 2. Charge cognitive

### Diagnostic

La progression est lisible, mais la charge cognitive peut devenir élevée pour des élèves B1/B1+ si chaque séance introduit :

- un nouveau support ;
- un nouveau concept culturel ;
- du lexique artistique ;
- une structure argumentative ;
- une interprétation critique.

Le risque principal est Lesson 5 : “connections, memory and community” peut ajouter une couche conceptuelle tardive alors que les élèves doivent déjà préparer le pitch.

### Correction prioritaire

Maintenir Lesson 5 seulement si elle sert explicitement la nuance finale.

Sinon :

- fusionner Lesson 5 avec Lesson 4 ;
- ou transformer Lesson 5 en homework / mini-research guidée ;
- ou l’utiliser comme différenciation pour élèves avancés.

Décision recommandée : garder Lesson 5 comme **séance courte de consolidation de la nuance**, pas comme nouvelle exploration documentaire lourde.

## 3. ZPD et degré de guidage

### Diagnostic

La séquence prévoit bien une montée progressive :

1. observer ;
2. classer ;
3. interpréter ;
4. justifier ;
5. nuancer ;
6. pitcher.

Mais les lesson plans devront absolument doser le guidage. En B1/B1+, demander trop tôt “Does art change stereotypes?” risque de provoquer des réponses vagues.

### Garde-fou

Chaque séance doit suivre une montée cognitive :

```text
observable clue → first inference → class recap → reusable language → micro-production
````

À éviter :

```text
What is the message of the artwork?
```

À préférer :

```text
Look at the artwork and list 3 visible elements.
Classify them: wall / people / message / colour / symbol.
What can we infer from these elements?
```

## 4. Statut de la worksheet

### Diagnostic

Le statut est bien défini dans `03_sequence.md` : worksheet = outil temporaire de repérage, classement, hypothèse, justification.

C’est conforme au principe selon lequel une activité d’accès au sens doit faire agir l’élève sur le document et non simplement lui faire répondre à des questions.

### Risque

Au moment de produire les worksheets, le risque sera de retomber dans :

* questions directes ;
* QCM ;
* vrai/faux ;
* “answer the questions” ;
* trace écrite pré-remplie.

### Garde-fou

Chaque worksheet devra contenir au moins une activité de type :

* classify;
* match;
* highlight evidence;
* complete a curator’s note;
* rank possible effects;
* justify your choice.

## 5. Statut du class recap

### Diagnostic

Le class recap est correctement pensé comme co-construit. Les formulations proposées sont utiles :

* At first, the border may look like...
* This artwork challenges the stereotype that...
* It helps visitors see...
* However, stereotypes may remain strong because...

### Point à renforcer

Le recap ne doit pas être seulement une phrase finale. Il doit devenir une **banque cumulative de formulations** pour le pitch.

### Correction recommandée

Créer une section récurrente dans chaque lesson plan :

```text
Reusable class recap for the final pitch
```

Exemple :

```text
Today’s sentence we can reuse in the final pitch:
This artwork challenges the stereotype that...
```

## 6. Rôle du diaporama, du tableau et du cahier

### Diagnostic

Le rôle des outils n’est pas encore assez défini.

### Décision recommandée

| Support de classe | Rôle                                                                                    |
| ----------------- | --------------------------------------------------------------------------------------- |
| Diaporama         | Montrer les documents visuels, guider l’observation, afficher les consignes courtes     |
| Tableau           | Construire le class recap avec les élèves, organiser les hypothèses, enrichir la langue |
| Cahier            | Fixer la trace utile pour la tâche finale : formulations, lexique, mini-recap           |
| Worksheet         | Support de travail temporaire, non trace finale                                         |

### Point de vigilance

Le diaporama ne doit pas contenir la trace écrite finale déjà rédigée. Sinon, on court-circuite la construction du sens.

## 7. Évaluation observable

### Diagnostic

La tâche finale est observable : pitch oral curatorial avec support visuel et courte interaction.

Les critères provisoires sont pertinents :

* projet culturel clair ;
* lien explicite avec la problématique ;
* explication des œuvres ;
* justification des choix ;
* nuance ;
* organisation ;
* intelligibilité ;
* interaction.

### Point à améliorer

La grille devra éviter de séparer artificiellement culture et langue. Par exemple, “qualité culturelle” et “anglais” ne doivent pas devenir deux colonnes indépendantes qui pénalisent deux fois les élèves fragiles.

### Critères recommandés pour future grille

1. **Cultural message**
   The project clearly explains how Border Art can change visitors’ perception.

2. **Use of evidence**
   The students refer to artworks / projects studied or selected.

3. **Curatorial justification**
   The students explain why they selected these artworks.

4. **Nuance**
   The students acknowledge that stereotypes may remain strong.

5. **Oral organisation**
   The pitch follows a clear structure.

6. **Language and intelligibility**
   The message is understandable and uses key structures from the sequence.

7. **Interaction**
   The students answer a short question coherently.

## 8. Numérique au service de la langue

### Diagnostic

Le support visuel final est justifié, mais il doit rester au service du pitch oral.

### Risque

Canva / Slides peut devenir décoratif : belles images, peu de langue, peu d’argumentation.

### Garde-fou

Limiter le support visuel final :

* 1 slide ou 1 mini-poster ;
* 2 ou 3 images maximum ;
* très peu de texte ;
* mots-clés seulement ;
* obligation d’expliquer oralement le lien entre image, message et stéréotype.

Le numérique a une valeur ajoutée uniquement s’il aide à médiatiser le message culturel.

## 9. Progression CECRL et spiralaire

### Diagnostic

La progression B1 → B1+ est plausible.

Les fonctions langagières reviennent correctement :

* describe;
* explain;
* justify;
* nuance;
* pitch.

Cela respecte une logique spiralaire : les structures sont réactivées et enrichies dans des contextes nouveaux, au lieu d’être traitées une seule fois.

### Point à renforcer

Prévoir un rituel court de réactivation au début des séances 2 à 6 :

```text
Last time, we learned that...
Today, we will use this idea to...
```

Ou en anglais :

```text
Last lesson, we saw the border as...
Today, we will see how art can...
```

## 10. Corrections prioritaires avant Lesson 1

Avant de produire les lesson plans, il faut :

1. verrouiller les supports exacts de Lessons 1 à 5 ;
2. décider si Lesson 5 reste une vraie séance ou devient une extension ;
3. fixer le format final du support visuel : slide unique, mini-poster ou oral avec images ;
4. préparer une banque de langue cumulative ;
5. prévoir dès Lesson 1 une première formulation autour de stereotype ;
6. interdire les worksheets de type questionnaire ;
7. réserver `90_audit.md` à l’audit didactique et non au simple rapport technique.

## 11. Verdict final

Architecture validée avec réserves.

Statut :

```text

Lesson plans : non encore autorisés sans verrouillage des supports exacts
Worksheets : non autorisées avant lesson plans
Grille : non autorisée avant stabilisation de la tâche finale
Slides : non autorisés avant définition du rôle du diaporama
```
